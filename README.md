# Building Block Lab

## Task Description

This program works with building blocks (`BuildingBlock` class) and implements the following tasks:

1. **Sorting**: Sorts an array of blocks based on:
   - **Primary field**: `hardness` (ascending order).
   - **Secondary field**: `weight` (descending order).
2. **Search**: Checks if a specific block (based on all its fields) exists in the array.

## How to Use

1. Clone this repository to your local machine:
2. Open the project in your preferred Java IDE.
3. Run the `Main` class.

## Requirements

- Java Development Kit (JDK) 8 or later.
- Internet connection to clone the repository.
